# 🚗 Comparador Gasolina vs Álcool

Aplicativo mobile desenvolvido com **Flutter** para ajudar motoristas a decidirem, de forma rápida e prática, qual combustível é mais vantajoso: **gasolina ou álcool (etanol)**.

---

## 📱 Preview

O app conta com:
- Tela de login
- Tela principal com cálculo de combustível
- Interface simples, direta e intuitiva

---

## 📊 Como funciona

O cálculo segue uma regra amplamente utilizada no Brasil:

> Divide-se o preço do álcool pelo preço da gasolina.

- Se o resultado for **≥ 0,70** → **Gasolina compensa mais**
- Se o resultado for **< 0,70** → **Álcool compensa mais**

---

## 🧮 Fórmula utilizada
resultado = preco_alcool / preco_gasolina

---

## 🎯 Objetivo

Facilitar a tomada de decisão no momento do abastecimento, eliminando a necessidade de cálculos manuais e tornando o processo mais rápido, prático e acessível.

---

## 🔐 Funcionalidades
